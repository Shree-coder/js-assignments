Various method in console function.

Types of console

Please open inspect window/console window in your browser to check!!!

  1. console.clear() : clear method "clears console windows"
  2. console.time('time') : starts calculate time required to work following commands
  3. console.log({a}) : prints the content inside the log() on console window
  4. console.error("There is an error") : show error message on console window of the browser
  5. console.warn("There is an warning") : shows warning message
  6. console.table({a}) : creating tabular form of a given array
  7. console.timeEnd('time') : time calculation stops here and display consumed time on console window
