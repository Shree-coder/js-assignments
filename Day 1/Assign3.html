Data Types:

datatypes
  1. Number : The number type represents both integer and floating point numbers.
  2. BigInt : to represent integers of arbitrary length.
  3. String : A string in JavaScript must be surrounded by quotes.
  4. Boolean : The boolean type has only two values: true and false.
  5. Null : In JavaScript null is "nothing".
  6. Undefined : In JavaScript, a variable without a value, has the value undefined.
  7. Objects : JavaScript objects are written with curly braces {}.
  8. Array : JavaScript arrays are written with square brackets. Array items are separated by commas.
